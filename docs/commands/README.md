# コマンド別ドキュメント

csv-ops の各サブコマンドの使い方。

| サブコマンド          | 概要                                                                  |
| --------------------- | --------------------------------------------------------------------- |
| [mask](mask.md)       | 指定カラムを文字数保持でマスクする（元の文字数分 `*` に置換）         |
| [replace](replace.md) | 指定列または全カラム横断で文字列を一括置換する（単純置換 / 正規表現） |
| [flag](flag.md)       | 指定列を正規表現で判定し、真偽値の列を末尾に追加する                  |
| [extract](extract.md) | 指定列からパターンマッチした文字列を抽出し、列を末尾に追加する        |

各ファイルは「書式 / オプション / 設定ファイル / 使用例 / 備考」で構成。
